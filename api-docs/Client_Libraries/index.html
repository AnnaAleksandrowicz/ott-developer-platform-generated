When developing applications that interact with the Kaltura API, it is best to use a native Client Library.

Please note, the client libraries provided here have been generated for version 4.1 of the Kaltura OTT Platform API. Customers are encouraged to verify their platform API version and ensure the use of client libraries that match the version of their platform version for optimal compatibility.
